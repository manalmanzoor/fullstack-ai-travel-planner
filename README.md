# 🌍 AI Travel Planner

An AI-powered full-stack travel planner that generates personalized travel itineraries based on your destination, trip duration, interests, and budget. The application combines a modern frontend with AI-powered itinerary generation to help users plan their trips quickly and efficiently.

---

## ✨ Features

- 🤖 AI-generated personalized travel itineraries
- 🗺️ Browse curated travel destinations
- 💰 Budget-based trip planning
- 📅 Day-wise itinerary generation
- 🎨 Modern responsive UI with animations
- ⚡ Fast AI responses using Groq
- 📱 Mobile-friendly design

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/manalmanzoor/fullstack-ai-travel-planner.git
```

### Navigate to the project

```bash
cd fullstack-ai-travel-planner
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Install backend dependencies

```bash
cd ../backend
npm install
```

---

## ▶️ Run the Application

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
cd ../frontend
npm run dev
```

Open:

```
http://localhost:5173
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
GROQ_API_KEY=your_groq_api_key
```
